📿 Prayer Beads Cost Calculator

A Python project I made to explore coding with a real idea.  
The program helps you design custom prayer using natural stones and then shows the total cost.

🧰 Tools used

- Python 3
- Terminal / command line

✨ What it does

- Shows simple menus for stones, strings, tassels, and quality  
- Lets you type in your choices as numbers  
- Uses lists to match each choice with a price  
- Adds everything together and prints the final price

🚦 Running the project

1. Clone this repository or download the ZIP from GitHub.  
2. Open a terminal in the project folder and run: `python main.py`
